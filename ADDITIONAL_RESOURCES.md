## Online Courses
- [edX: Introduction to Kubernetes](https://www.edx.org/course/introduction-to-kubernetes) — Free course by The Linux Foundation
- [Coursera: Getting Started with Google Kubernetes Engine](https://www.coursera.org/learn/google-kubernetes-engine) — GKE-focused beginner course
- [Udemy: Kubernetes for the Absolute Beginners](https://www.udemy.com/course/kubernetes-for-beginners/) — Hands-on beginner course by Mumshad Mannambeth
- [Udemy: Certified Kubernetes Administrator (CKA)](https://www.udemy.com/course/certified-kubernetes-administrator-with-practice-tests/) — CKA exam prep with practice labs
- [Linux Foundation: Kubernetes Fundamentals (LFS158)](https://training.linuxfoundation.org/training/introduction-to-kubernetes/) — Free introductory course

## Books
- **Kubernetes in Action** by Marko Lukša — Deep dive into Kubernetes internals and patterns
- **The Kubernetes Book** by Nigel Poulton — Beginner-friendly, concise overview
- **Kubernetes Patterns** by Bilgin Ibryam & Roland Huß — Design patterns for cloud-native apps
- **Production Kubernetes** by Josh Rosso et al. — Enterprise-grade Kubernetes in practice

## Certifications
- [CKA – Certified Kubernetes Administrator](https://www.cncf.io/certification/cka/) — Core admin skills exam
- [CKAD – Certified Kubernetes Application Developer](https://www.cncf.io/certification/ckad/) — Developer-focused exam
- [CKS – Certified Kubernetes Security Specialist](https://www.cncf.io/certification/cks/) — Security-focused advanced exam

## Tools and Ecosystem
- [Helm](https://helm.sh/) — The package manager for Kubernetes
- [k9s](https://k9scli.io/) — Terminal-based UI for managing Kubernetes clusters
- [Lens](https://k8slens.dev/) — Desktop IDE for Kubernetes
- [Kustomize](https://kustomize.io/) — Native Kubernetes configuration management
- [Skaffold](https://skaffold.dev/) — Local Kubernetes development workflow tool
- [Minikube](https://minikube.sigs.k8s.io/) — Run Kubernetes locally
- [Kind](https://kind.sigs.k8s.io/) — Run Kubernetes clusters using Docker containers
