# Kubernetes Learning Notebook

This repository contains my personal learning journey with Kubernetes, covering topics from the fundamentals of container orchestration through advanced subjects including storage, security, observability, autoscaling, advanced scheduling, stateful workloads, Helm package management, and CI/CD with GitOps. It includes commonly used commands and notes based on real hands-on experience.

## How to Use This Notebook
Read the modules in order — each one builds on the previous. Use `COMMANDS_REFERENCE.md` as a quick-lookup cheat sheet while practicing.
